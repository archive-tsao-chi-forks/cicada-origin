# Syntax

## Syntax of expression

| Expression | Example                                        |
|------------|------------------------------------------------|
| pi type    | `(x: t) -> body[x]`, `forall(x: t) -> body[x]` |
| function   | `(x: t) => body[x]`                            |
| sigma type | `exists(x: t, ..., body[x, ...])`              |
| pair       | `tuple(x, ..., z)`                             |
