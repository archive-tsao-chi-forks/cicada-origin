# 蝉语 / Cicada Language

## 使用 / Usage

### 命令行工具 / Command Line Tools

- 安装 / Install: `npm install @cicada-lang/cicada --global`
- 命令 / Commands:
  ```
  cic check-library <config-file>  Check all files in a library
  cic run <file>                   Run a file
  ```
